# 숫자 야구

1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임

## 기능정의 목록

### 게임시작 기능

[] "숫자 야구 게임을 시작합니다."라는 문구를 출력한다.

### 입력받는 기능

[] console의 값을 입력받는다.

### 유효성 처리 기능

[] 입력값이 잘못된 경우 throw 문을 사용해 예외를 발생시킨후 애플리케이션을 종료한다.
[] 입력값은 중복이 아닌 값만 가능하다.
[] *입력값은 숫자만 가능하다.
[] *입력값은 길이가 3인 경우만 가능하다.

### 랜덤한 숫자를 생성하는 기능

[] 1에서 9까지 서로 다른 임의의 수 3개를 생성한다.

### 숫자를 비교하는 기능

[] 3개 숫자를 모두 맞추면 "3개의 숫자를 모두 맞히셨습니다! 게임 종료"를 출력한다.
[] 3개 숫자를 모두 맞추면 게임이 종료된다.
[] 같은 수가 전혀 없으면 '낫싱'이라는 문구을 출력한다.
[] 같은 수가 같은 자리에 있으면 '스트라이크'라는 문구를 출력한다.
[] 같은 수가 다른 자리에 있으면 '볼'이라는 문구를 출력한다.
[] 볼, 스트라이크 개수를 출력한다.

### 게임종료 기능

[] "게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요."라는 문구를 출력한다.
[] 게임 종료 후 게임을 다시 시작한다.
[] 게임을 완전히 종료한다.
[] 입력값이 잘못된 경우 throw 문을 사용해 예외를 발생시킨후 애플리케이션을 종료한다.
