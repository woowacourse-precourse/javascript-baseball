# 기능 나누기

## 1. 게임 시작 함수

- 2번과 3번 함수를 호출한다.

## 2. 랜덤으로 서로 다른 숫자 3개 골라 놓기 함수

- 문제에서 주어진 Random 함수를 사용하여 1~9 사이의 숫자를 저장해 놓는다.

## 3. 사용자 입력 함수

- 문제에서 주어진 Console.readline 함수를 사용하여 사용자의 추측 답안을 받아 놓는다.

## 4. 입력 유효 검사 함수

- 숫자만 받았는지 확인한다.
- 1 ~ 9 사이의 숫자인지 확인한다.
- 3자리의 숫자인지 확인한다.
- 같은 숫자가 포함이 안됐는지 확인한다.

## 5. 사용자 숫자와 랜덤 답안 숫자 비교 함수

1. 사용자가 입력한 숫자와 랜덤 답안 숫자를 Object로 만들어 현재 숫자의 위치와 값을 함께 저장한다.
2. JSON.stringify를 사용하여 같은 숫자인지 비교한 뒤 같으면 strike를 3으로 바꾼뒤 [스트라이크, 볼카운트]를 return한다.
3. 2번에서 서로 다른 숫자인 것을 확인 했다면 map함수 안에서 indexOf를 통해서 현재의 위치값과 타겟의 위치가 같다면 strike + 1, 다르다면 ballCount + 1을 한 뒤 3자리 모두 비교가 끝나면 [strike, ballCount]를 return 한다.

## 6. 스트라이크 볼 출력 함수

- 5번 함수에서 return 받은 값을 받아와서 if - else 문을 통해서 함수 출력
- 스트라이크가 3개인 경우 7번 재시작 함수를 호출한다.
- 스트라이크가 3개가 아닌 경우 3번 함수를 다시 호출한다.

## 7. 재시작 함수

- 게임을 재시작 할 것인지 아닌지 1번 2번을 통해 정한다.
- 1번을 골랐을 경우 1번 함수를 다시 호출한다.
- 2번을 골랐을 경우 Console.close()를 통해서 프로그램을 종료한다.
