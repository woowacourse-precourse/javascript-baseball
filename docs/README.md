## 기능 목록

#### 1~9 사이의 서로 다른 임의의 수 3개를 뽑는다.

- MissionUtils 라이브러리 사용

#### 플레이어가 서로 다른 3개 숫자를 입력받는다.

- 숫자를 입력받고 엔터를 누르면 제출
- 조건에 맞지 않는 숫자를 누르면 다시 입력
- 서로 다른 숫자여야 한다.
- 1~9사이의 숫자여야한다.
- 개수가 3개여야한다.
- 음수가 나오면 안된다.

#### 플레이어가 입력한 숫자에 대한 결과 출력

- 같은 수가 자리에 있으면 스트라이크
- 다른 자리에 있으면 볼
- 같은 수가 없으면 낫싱

#### 컴퓨터 숫자를 플레이어가 맞추면 게임 종료

- 3 스트라이크
