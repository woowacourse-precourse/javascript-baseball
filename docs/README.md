### 기능 목록

1. 프로그램 실행
2. 정답 숫자 선정
    - 1~9 사이의 숫자인가
    - 각각 다른 숫자로 구성되었는가
3. 사용자로부터 입력값 얻기
    - 모두 숫자인가
    - 1~9 사이의 숫자인가
    - 3자리의 숫자인가
    - 각각 다른 숫자로 구성되었는가
4. 사용자가 잘못된 값을 이용한 경우 예외 처리
5. 정답 숫자와 사용자의 입력값 체크
6. 결과 출력
7. 프로그램 종료 및 재시작

### 리팩토링 목록

1. 객체지향 프로그래밍을 했는가
2. 자바스크립트 컨벤션을 지켰는가
3. 함수가 한 가지 일만 하고 있는가
