## 기능 구현

-[] 시작 후 메세지 출력 => Console.print(), `숫자 야구 게임을 시작합니다.`
- 여기서부터 while
-[] 랜덤 변수 받기 => Random.pickNumberInRange()
-[] 숫자 입력 => Console.readLine()
    -[] 입력 숫자에 대한 처리
        -[] 입력에 대한 유효성 검사
            -[] 서로 다른 숫자
            -[] 3개의 숫자
            -[] 1~9 내 숫자
        -[] 입력 숫자에 대한 확인 함수 실행
            -[] 같은 자리, 같은 숫자 스트라이크
            -[] 다른 자리, 같은 숫자 볼
        -[] 결과 출력
            -[] 볼, 스트라이크 개수 출력 => `x볼 y스트라이크`
            -[] 낫싱 출력 => `낫싱`
-[] 종료 후 메세지 출력 => `3개의 숫자를 모두 맞히셨습니다! 게임 종료`
-[] 게임 종료 후 숫자 입력 => `게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.`
    -[] 1, 2가 아닌 숫자 입력시 예외처리
        -[] 1인 경우 재시작
        -[] 2인 경우 종료
- 여기까지 while
