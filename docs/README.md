# 미션 - 숫자 야구

## 🎯 기능 구현 목록

0. 게임 스타트 문구 출력
   - "숫자 야구 게임을 시작합니다." 문구를 출력
1. 사용자는 세자리 숫자를 입력한다.
   - "세자리 숫자를 입력해주세요 : " 문구와 함께 입력을 받는다.
   - [예외처리] 입력값이 세자리인지 아닌지 검증한다.
   - [예외처리] 입력값이 1부터 9사이의 값인지 검증한다.
   - [예외처리] 입력값이 서로 다른 수인지 검증한다.
2. 사용자가 입력한 숫자를 상대방의 숫자와 비교한다.
   - 상대방의 숫자는 랜덤값으로 주어진다.
   - 각 숫자의 자릿수별로 비교하여 출력값을 결정한다.
     - 사용자가 상대방의 숫자와 그 위치를 맞추면 스트라이크를 저장
     - 사용자가 상대방의 숫자를 맞추고 위치를 맞추지 못하면 볼을 저장
     - 모든 숫자에 대해서 사용자가 상대방의 숫자와 그 위치를 모두 맞추지 못하면 낫싱 반환
3. 숫자를 비교한 뒤 결과를 출력한다.
   - 입력한 수에 대한 결과를 볼, 스트라이크 개수로 출력한다.
   - 볼과 스트라이크가 하나도 없는 경우에는 낫싱을 출력한다.
4. 재시작 여부 확인
   - 3스트라이크가 출력이 안될 경우
     - 1번~3번의 과정을 반복한다.
   - 3스트라이크가 출력됐을 경우
     - "3개의 숫자를 모두 맞히셨습니다! 게임 종료" 문구 출력
     - "게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요." 문구 출력
     - 1과 2중에 한 숫자를 입력한다.
       - 1을 입력할 경우, 1번으로 돌아간다.
       - 2를 입력할 경우, 게임이 종료된다.
       - [예외처리] 입력값이 1과 2중 하나인지 검증한다.
