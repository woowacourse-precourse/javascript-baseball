## 구현 목록

- [ ] Game Input 받기
  - [ ] Input(예외 처리) 3자리 인지 or 1-9 값인지 or 중복된 값이 있는지
  - [ ] 잘못된 값 들어오면 (throw 활용해서 프로그램 종료)
- [ ] 유저 Input 받기
  - [ ] 유저 Input 값 (1 = New Game 2 = 종료)
    - [ ] 예외 값 (throw 활용해서 프로그램 종료)
- [ ] Game Input 값 과 유저 Input 비교 해서 출력
  - [ ] 3자리 수 다 맞추면 유저 Input 받기 첫번째 구현으로 넘어감
