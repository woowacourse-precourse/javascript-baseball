## 📚 기능 목록

1. 컴퓨터의 무작위 숫자 값 추출 (🔴) (🔵)

- 숫자를 string으로 변환 (🔴)

2. 사용자로부터 입력받기 (🔴)

3. 숫자를 배열화 (🔴) (🔵)

4. 사용자의 입력 값 유효 (🔵)

- ⚠️ 빈칸이 아닌 1~9 사이의 숫자만을 입력했는가? (🔴) (🔵)
- ⚠️ 숫자 3개를 입력했는가? (🔴) (🔵)
- ⚠️ 중복값이 존재하나? (🔴) (🔵)

5. 사용자가 입력을 잘못했을 때 에러발생 (🔴) (🔵)

6. 사용자의 입력값과 컴퓨터의 무작위 값 비교 후 전달(게임룰을 기반으로) (🔴) 🔵

7. 3스트라이크가 나왔을 때 재시작/종료 숫자로 입력받기 (🔴)

8. 입력 받은 숫자의 유효성 체크 (🔴) (🔵)

- 1,2 중 한가지를 입력했나? (🔵)

- 숫자를 1개만 입력했나? (🔵)

9. 유저의 재시작/종료 선택에 따라 게임 재시작 혹은 종료 (🔴)
