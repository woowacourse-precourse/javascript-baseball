## 숫자야구 기능 목록

1. 상대방(컴퓨터, 정답) 숫자 ramdom 생성

- 1부터 9까지
- 서로 다른 숫자
- 세 자리

2. 사용자가 입력하는 숫자 input

3. 숫자야구 게임 진행

- 스트라이크 : 같은 수가 같은 자리에 위치
- 볼 : 같은 수가 다른 자리에 위치
- 낫싱 : 같은 수가 전혀 없을 때

- 이후에 각 결과 횟수 counting

4. 게임 결과 문구 출력 output

5. 게임 종료 및 새로 시작 조건 출력

- 새로 시작 : 1
- 게임 종료 : 2
