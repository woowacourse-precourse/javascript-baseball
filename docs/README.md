구현할 기능 목록
1. computer에서 랜덤 숫자 생성
* 숫자 범위 1 ~ 9
* 중복 없이 3자리 숫자 생성

2. user에게 숫자 입력 받기
* 숫자 범위 1 ~ 9
* 중복 없이 3자리 숫자 받기
* 잘못된 값이 있다면 throw로 예외 발생 후 종료

3. 정답 체크
* strike, ball, 낫싱 확인

3-1. 정답일 때
* 게임 종료
* 재시작 / 종료를 1 / 2로 구분

3-2. 정답이 아닐 때
* strike와 ball의 카운트 띄우기
* 같은 수가 같은 자리에 있는 경우 : strike
* 같은 수가 다른 자리에 있는 경우 : ball
* 같은 수가 없는 경우 : 낫싱
