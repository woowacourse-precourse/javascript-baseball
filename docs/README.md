# 🚀 기능 목록

## 램덤 숫자 3개 뽑기

- MissionUtils 라이브러리의 Random.pickNumberInRange()를 활용
- 중복된 숫자 제거

<br/>

## 사용자 입력 받기

- MissionUtils 라이브러리에서 제공하는 Console.readLine를 활용

<br/>

## 사용자 입력의 유효성 검사

- 입력이 숫자인가
- 입력의 크기가 3인가
- 숫자 범위가 1~9인가
- 숫자 중복이 없는가

<br/>

## 스트라이크 구하기

- strike 멤버 변수 업데이트

<br/>

## 볼 구하기

- ball 멤버 변수 업데이트

<br/>

## 결과 출력하기

- MissionUtils 라이브러리에서 제공하는 Console.print를 활용

<br/>

## 게임 재시작

- 1은 재시작, 2는 종료

<br/>
