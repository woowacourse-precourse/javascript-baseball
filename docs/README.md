- 결과에 따라서 반복할지 재시작할지 판단
- 입력이 비동기이기 때문에 재귀함수로 구현

### 2. 사용자 입력을 검사하는 기능

- 유효하지 않을 경우 에러를 발생

### 3. 사용자 입력 배열을 만드는 기능

### 4. 3개의 난수 배열을 만드는 기능

- 중복이 아닌 난수만 추가하여 저장

### 5. 볼과 스트라이크를 계산하는 기능

### 6. 출력 문자열을 생성하는 기능

- 볼과 스트라이크 개수에 따라서 문자열 생성

### 7. 재시작하거나 종료하는 기능
