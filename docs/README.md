## 🔨 기능 목록

- 게임시작 멘트를 출력하는 기능
- 컴퓨터가 숫자를 정하는 기능
- 사용자로부터 숫자를 입력받는 기능
- 입력된 유저의 수를 숫자 배열로 변환하는 기능
- 입력한 수와 컴퓨터 수를 비교하는 기능
- 입력한 수에 대한 결과를 나타내는 기능
- 답을 맞췄는지 확인하는 기능
- 게임이 종료될 때 재시작과 종료를 물어보는 기능
- 게임을 재시작이나 종료하는 기능
- 잘못된 값을 입력한 경우 예외를 처리하는 기능
    - 입력된 숫자가 3개인지
    - 똑같은 숫자가 없는지
    - 0을 포함하고 있지 않은지
    - 숫자가 아닌지
